# Игра "Змейка" для TouchDesigner

Этот проект содержит полную реализацию классической игры "Змейка" для TouchDesigner с двумя вариантами отображения: текстовым и графическим.

## Файлы проекта

- `snake_game.py` - Основной код игры с логикой и текстовым отображением
- `touchdesigner_setup.py` - Дополнительные компоненты и графическая версия
- `README_Snake_TouchDesigner.md` - Данная инструкция

## Быстрая настройка (Текстовая версия)

### Шаг 1: Создание Text DAT
1. В TouchDesigner создайте **Text DAT**
2. В параметрах Text DAT установите:
   - `DAT` → `Text` → введите следующий код:
```python
# Импортируем основной модуль игры
exec(open('snake_game.py').read())

# Основная функция для отображения
me.text = get_display_text()
```

### Шаг 2: Создание Timer CHOP
1. Создайте **Timer CHOP**
2. Установите параметры:
   - `Timer Type`: Timer
   - `Length`: 0.5 (скорость игры)
   - `Auto Start`: On

### Шаг 3: Создание Script DAT для обновления игры
1. Создайте **Script DAT**
2. Подключите Timer CHOP к Script DAT
3. В Script DAT введите:
```python
def onPulse(channel):
    """Вызывается каждый раз при срабатывании таймера"""
    # Находим Text DAT с игрой (замените 'text1' на имя вашего Text DAT)
    text_dat = op('text1')
    
    # Обновляем игру
    exec(open('snake_game.py').read())
    text_dat.text = update_game()

def onValueChange(channel, sampleIndex, val, prev):
    pass
```

### Шаг 4: Создание обработки клавиш
1. Создайте **Keyboard In CHOP**
2. В параметрах включите клавиши: `up`, `down`, `left`, `right`, `r`
3. Создайте еще один **Script DAT**
4. Подключите Keyboard In CHOP к новому Script DAT
5. В этот Script DAT введите:
```python
def onValueChange(channel, sampleIndex, val, prev):
    """Обработка нажатий клавиш"""
    if val == 1.0:  # Клавиша нажата
        key_name = channel.name
        
        # Импортируем игру
        exec(open('snake_game.py').read())
        
        # Обрабатываем клавиши
        if key_name == 'up':
            game.change_direction((0, -1))
        elif key_name == 'down':
            game.change_direction((0, 1))
        elif key_name == 'left':
            game.change_direction((-1, 0))
        elif key_name == 'right':
            game.change_direction((1, 0))
        elif key_name == 'r':
            game.reset_game()
        
        # Обновляем отображение (замените 'text1' на имя вашего Text DAT)
        text_dat = op('text1')
        text_dat.text = game.get_game_field()

def onPulse(channel):
    pass
```

## Управление игрой

- **↑↓←→** - Управление направлением движения змейки
- **R** - Перезапуск игры
- **Цель** - Собирать еду (*) и избегать столкновений

## Графическая версия (Дополнительно)

Для создания графической версии с цветным отображением:

### Шаг 1: Создание Script TOP
1. Создайте **Script TOP**
2. В параметрах установите разрешение (например, 400x300)
3. В Script TOP введите код из `script_top_code` в файле `touchdesigner_setup.py`

### Шаг 2: Настройка рендеринга
Графическая версия отображает:
- **Голову змейки** - красным цветом
- **Тело змейки** - зеленым цветом  
- **Еду** - желтым цветом

## Параметры игры

В классе `SnakeGame` можно настроить:
- `width` - ширина игрового поля (по умолчанию 20)
- `height` - высота игрового поля (по умолчанию 15)
- Скорость через параметр `Length` в Timer CHOP

## Структура кода

### Основные методы класса SnakeGame:
- `reset_game()` - Сброс игры к начальному состоянию
- `generate_food()` - Генерация еды в случайном месте
- `change_direction()` - Изменение направления движения
- `move()` - Логика движения и проверки столкновений
- `get_game_field()` - Формирование текстового представления игры

### Обработчики TouchDesigner:
- `onPulse()` - Вызывается по таймеру для обновления игры
- `onValueChange()` - Обрабатывает нажатия клавиш
- `onKeyboardEvent()` - Альтернативный обработчик клавиш

## Возможные улучшения

1. **Звуковые эффекты** - добавить Audio File In DAT для звуков поедания еды и окончания игры
2. **Анимации** - использовать Math CHOP для плавных переходов
3. **Меню** - создать стартовое меню с кнопками
4. **Уровни сложности** - изменение скорости в зависимости от счета
5. **Рекорды** - сохранение лучших результатов в Table DAT

## Отладка

Если игра не работает:
1. Проверьте пути к файлам в `exec(open('snake_game.py').read())`
2. Убедитесь, что Timer CHOP правильно подключен к Script DAT
3. Проверьте имена операторов в `op('text1')`
4. Включите отображение ошибок в Text Port
